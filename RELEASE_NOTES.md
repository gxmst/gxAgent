# gxAgent v1.0.1

## 🎯 稳定版本

这是一个稳定、安全的版本，包含完整功能但**不包含并行执行**（为保证 MCP 工具和安全检查点正常工作）。

## ✨ 功能特性

- ✅ 会话导入/导出
- ✅ 自定义右键菜单（全局 + 会话）
- ✅ 工具调用统计
- ✅ 命令联想（输入 `/`）
- ✅ 快捷键（Ctrl+K 新建对话）
- ✅ 自动保存草稿
- ✅ 工具超时 5 分钟
- ✅ 自定义图标

## 📦 下载

### Windows 安装包（推荐）
- 自动安装 WebView2
- 创建快捷方式和开始菜单项
- 支持完整卸载

### Windows 便携版
- 单 exe 文件
- 需要手动安装 [WebView2](https://go.microsoft.com/fwlink/?linkid=2124701)

## 🔧 系统要求

- Windows 10/11 (64位)
- Edge WebView2 运行时
