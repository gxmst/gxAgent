## 💡 关于标题栏

Windows 原生标题栏颜色限制：
- **系统控制**：颜色由 Windows 主题决定
- **Tauri 限制**：无法直接修改原生标题栏颜色
- **替代方案**：使用 Overlay 模式（已应用）

如果需要完全自定义标题栏：
```json
{
  "decorations": false  // 隐藏原生标题栏
}
```
然后自己用 HTML/CSS 绘制（需要添加拖动区域等功能）
