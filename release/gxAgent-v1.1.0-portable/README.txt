# gxAgent v1.1.0 便携版

## 运行要求

1. **操作系统**：Windows 10/11 (64位)
2. **运行时依赖**：Edge WebView2

## 首次运行

### 如果双击 gxAgent.exe 后：

**✅ 正常打开** → 无需操作，开始使用

**❌ 报错或白屏** → 需要安装 WebView2：
1. 下载：https://go.microsoft.com/fwlink/?linkid=2124701
2. 运行安装程序
3. 重新打开 gxAgent.exe

## 使用说明

- 双击 `gxAgent.exe` 启动
- 配置文件保存在：`C:\Users\你的用户名\.gxagent\`
- 可放到任意目录运行
